## For SPA
````
git clone --bare https://github.com/CognizantStudio/lp-auth-react-app.git
cd lp-auth-react-app.git
git push --mirror https://github.com/CognizantStudio/<<your-new-repo>>.git
cd ..
del lp-auth-react-app.git
````
